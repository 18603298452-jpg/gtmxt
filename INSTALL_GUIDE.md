# The Freak Circus - iOS 安装指南

## 📱 应用简介

**The Freak Circus（怪胎马戏团）** 是一款心理恐怖视觉小说游戏。本项目将网页游戏打包为iOS原生WebView应用，支持全屏显示，提供更好的游戏体验。

- **游戏名称**: The Freak Circus
- **游戏类型**: 心理恐怖视觉小说
- **网页地址**: https://playthefreakcircus.com
- **最低系统要求**: iOS 14.0 或更高版本
- **支持设备**: iPhone (推荐)、iPad

## 📦 方案一：GitHub Actions 云编译 + Sideloadly 签名（推荐）

这是最推荐的方案，完全免费，无需Mac，体验最好。

### 第一部分：使用 GitHub Actions 云编译 IPA

#### 1. 准备工作

- 注册一个 [GitHub 账号](https://github.com/)（免费）
- 下载本项目的所有文件

#### 2. 创建 GitHub 仓库

1. 登录 GitHub，点击右上角 **+** → **New repository**
2. 填写仓库名称（例如：`TheFreakCircus-iOS`）
3. 选择 **Public** 或 **Private** 都可以
4. 点击 **Create repository**

#### 3. 上传项目文件

**方法一：网页上传（简单）**
1. 在仓库页面点击 **uploading an existing file**
2. 将本项目的所有文件拖拽到上传区域
3. 点击 **Commit changes**

**方法二：Git 命令行（推荐）**
```bash
# 初始化仓库
git init
git add .
git commit -m "Initial commit"

# 添加远程仓库
git remote add origin https://github.com/你的用户名/TheFreakCircus-iOS.git

# 推送代码
git branch -M main
git push -u origin main
```

#### 4. 触发云编译

1. 进入你的 GitHub 仓库页面
2. 点击顶部的 **Actions** 标签
3. 左侧选择 **Build iOS IPA**
4. 点击右侧的 **Run workflow** → 选择 `main` 分支 → 点击 **Run workflow**
5. 等待编译完成（大约需要 5-10 分钟）

#### 5. 下载编译好的 IPA

1. 编译完成后，点击进入该次运行的详情页
2. 滚动到页面底部的 **Artifacts** 区域
3. 点击 **The-Freak-Circus-IPA** 下载压缩包
4. 解压后得到 `The-Freak-Circus-unsigned.ipa` 文件

> 💡 **提示**: GitHub Actions 对公开仓库完全免费，私有仓库每月有 2000 分钟的免费额度，足够使用。

### 第二部分：使用 Sideloadly（Windows版）签名安装

Sideloadly 是一款免费的iOS应用签名工具，支持Windows和Mac，使用普通Apple ID即可签名。

#### 1. 下载 Sideloadly

- 官方网站: https://sideloadly.io/
- 点击 **Download for Windows** 下载安装包
- 支持 Windows 10/11

#### 2. 安装 Sideloadly

1. 运行下载的安装程序
2. 按照安装向导完成安装
3. 首次运行可能需要安装 iTunes 或 Apple Application Support
   - 如果提示缺少组件，请按照提示下载安装

#### 3. 准备工作

- 确保你的iPhone已通过数据线连接到电脑
- 手机上点击 **信任此电脑**
- 准备一个 Apple ID（普通账号即可，无需开发者账号）

> ⚠️ **注意**: 建议使用一个不常用的Apple ID进行签名，不要使用主账号。

#### 4. 签名并安装

1. 打开 Sideloadly
2. 在 **iDevice** 下拉菜单中选择你的iPhone
3. 点击 **IPA** 旁边的图标，选择下载好的 `The-Freak-Circus-unsigned.ipa` 文件
4. 在 **Apple ID** 栏输入你的Apple ID邮箱
5. 点击 **Start** 开始签名和安装
6. 输入你的Apple ID密码（如果开启了双重认证，需要输入应用专用密码）

#### 5. 信任开发者证书

安装完成后，还需要在手机上信任证书：

1. 打开 iPhone 的 **设置** → **通用** → **VPN与设备管理**
2. 在 **开发者APP** 下面找到你的Apple ID对应的证书
3. 点击进入，然后点击 **信任 "你的Apple ID"**
4. 确认信任

#### 6. 开始游戏

现在你就可以在桌面上找到 **The Freak Circus** 应用，点击开始游戏了！

### Sideloadly 常见问题

**Q: 签名的应用可以用多久？**
A: 普通Apple ID签名的应用有效期为7天，7天后需要重新签名安装。付费开发者账号有效期为1年。

**Q: 提示"Apple ID disabled"怎么办？**
A: 这是苹果的安全机制，换一个Apple ID试试，或者等待一段时间再试。

**Q: 安装失败，提示各种错误？**
A: 
- 确保手机已正确连接并信任电脑
- 尝试更换USB接口或数据线
- 确保手机系统版本满足要求（iOS 14.0+）
- 重启手机和电脑后重试

**Q: 如何设置应用专用密码？**
A: 
1. 访问 https://appleid.apple.com/
2. 登录你的Apple ID
3. 找到 **安全性** → **App 专用密码**
4. 点击 **生成专用密码**，按照提示操作

## 📌 方案二：Web Clip（添加到主屏幕）

这是最简单的方案，不需要任何电脑，直接在iPhone上操作即可。

### 操作步骤

1. 在 iPhone 上打开 **Safari 浏览器**
2. 访问游戏网址: **https://playthefreakcircus.com**
3. 等待网页加载完成
4. 点击屏幕底部的 **分享按钮**（方框加向上箭头的图标）
5. 向下滑动，找到并点击 **添加到主屏幕**
6. 可以修改名称为 "The Freak Circus"
7. 点击右上角的 **添加**

| 全屏体验 | ⭐⭐⭐ 基本全屏 | ⭐⭐⭐⭐⭐ 完美全屏 |

## ❓ 常见问题

### 游戏相关

**Q: 游戏进度会丢失吗？**
A: 游戏进度保存在网页的localStorage中，只要不卸载应用或清除数据，进度就不会丢失。

**Q: 可以离线玩吗？**
A: 不可以，这是网页游戏，需要联网才能运行。

**Q: 游戏卡顿怎么办？**
A: 
- 确保网络连接良好
- 关闭后台其他应用
- 尝试重启应用

### 安装相关

**Q: 7天到期后怎么办？**
A: 使用Sideloadly重新签名安装即可，游戏进度不会丢失。

**Q: 可以永久签名吗？**
A: 
- 如果你的设备支持TrollStore（iOS 14.0 - 16.6.1 部分版本），可以实现永久签名
- 购买苹果开发者账号（$99/年），签名有效期1年
- 使用企业证书签名（不推荐，容易被吊销）

**Q: 会被封号吗？**
A: 使用自己的Apple ID正常签名安装不会封号，这是苹果允许的正常操作。
